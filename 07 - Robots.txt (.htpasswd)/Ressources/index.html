Unrestricted access to htpasswd

The robots.txt file is meant to prevent robots from accessing some folders and file, but it also reveals their location, and if the access is not protected, it is easy to collect sensitive datas.

When inspecting the robots.txt file, we can see that it mentions two directories: User-agent: * Disallow: /whatever Disallow: /.hidden
Inside the /whatever folder, we can find a file named .htpasswd. It contains an username and a password: root:8621ffdbc5698829397d97767ac13db3
The password is hashed in md5, and it is reversed for "dragon".
This credentials can be used to connect to the admin area of the website, accessible at: http://192.168.56.101/admin/
