Unverified form

A form is a quick and easy way to allow a user to send data to a website, but it must be protected and the data sent must be verified, which is unlikely on this website.

On the Survey page, we can vote for peoples by assigning them a mark between 1-10.
This form is unfortunately not protected, and you can send any number that you want. There is no server side validation.
